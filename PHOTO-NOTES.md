# Headshot files — what's what

- **headshot.jpg** — the version LIVE on the website (compressed for fast loading). Currently: navy blazer, hair forward, diamond cross (from headshot-cross.png).
- **headshot-cross.png** — THE APPROVED FINAL (July 19, 2026): blazer + hair forward + diamond cross pendant. Made in one edit from headshot-blazer.png to keep the face faithful.
- **headshot-blazer.png** — first AI blazer edit (hair back, no cross). Face approved.
- **headshot-blazer-hair.png / headshot-final.png / headshot-final-v2.png** — experiments; the last two had face drift (each AI generation on top of another drifts further from the real face — always regenerate from an approved version, not from a drifted one).
- **headshot-original.jpg** — cleaned-up real photo (background removed, soft green backdrop).
- **Sue's Headshot.jpg** — the untouched original photo.

## To change the site photo
Overwrite `headshot.jpg` with any image (keep the name), then commit and push. The site updates automatically in ~1 minute.

## How the blazer edit was made
Photoshop 2026 → Home screen → "Prompt to edit" → Select image → type prompt → Generate (~20 Firefly credits per generation) → Download.
